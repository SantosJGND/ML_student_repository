# Machine Learning Engineering — Student Repository

ML learning materials covering supervised learning, Docker, FastAPI, and end-to-end ML projects.

## Structure

| Directory | Description |
|-----------|-------------|
| `intro_docker/` | Docker fundamentals (Dockerfile, compose, app) |
| `intro_fastapi/` | FastAPI fundamentals (API app) |
| `ml_exercises/` | 9 supervised learning notebooks (linear regression → PyTorch NN) + datasets |
| `ml_project_showcase/` | Full ML pipeline: ingestion, validation, training, registry, API deployment |
| `ml_project_templates/` | Reusable project templates (breast_cancer, heart_disease, income_project, etc.) |

## Setup

```bash
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
```
